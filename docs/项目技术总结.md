# 工具导航项目技术总结

## 📋 项目概述

这是一个基于 Next.js 的工具导航网站，采用现代化的前端技术栈，实现了分类浏览、筛选和工具展示功能。项目完全使用单文件架构，便于学习和理解。

## 🛠️ 技术栈详解

### 1. 核心框架与技术

#### Next.js 15.5.6 (App Router)
- **特点**: 使用最新的 App Router 模式，替代传统的 Pages Router
- **优势**: 
  - 更好的性能优化
  - 内置的服务端渲染 (SSR)
  - 自动代码分割
  - 文件系统路由

#### TypeScript
- **类型定义**: 使用接口定义数据结构
- **类型安全**: 编译时类型检查，减少运行时错误
- **代码提示**: 更好的开发体验和代码智能提示

#### Tailwind CSS
- **原子化CSS**: 使用预定义的工具类快速构建UI
- **响应式设计**: 内置断点系统 (sm, md, lg)
- **主题一致性**: 统一的设计系统和颜色规范

### 2. React 核心概念应用

### 3. 数据结构设计

#### 类型定义
```typescript
interface Tool {
  id: string;        // 唯一标识
  name: string;      // 工具名称
  description: string; // 工具描述
  url: string;       // 访问链接
}

interface SubCategory {
  id: string;
  name: string;
  tools: Tool[];     // 工具列表
}

interface Category {
  id: string;
  name: string;
  icon: string;      // Emoji 图标
  subCategories: SubCategory[];
}
```

#### 数据组织
- **层级结构**: 一级分类 → 二级分类 → 工具列表
- **关联关系**: 通过 ID 建立数据关联
- **扩展性**: 易于添加新分类和工具

## 🔧 核心功能实现

### 1. 状态管理

#### 状态设计
```typescript
// 当前选中的一级分类
const [selectedCategory, setSelectedCategory] = useState<string>('dev');

// 当前选中的二级分类  
const [selectedSubCategory, setSelectedSubCategory] = useState<string>('frontend');
```

#### 派生状态
```typescript
// 根据选中分类计算当前数据
const currentCategory = toolsData.find(cat => cat.id === selectedCategory);
const currentSubCategories = currentCategory?.subCategories || [];
const currentTools = currentSubCategories.find(sub => sub.id === selectedSubCategory)?.tools || [];
```

### 2. 事件处理

#### 分类切换逻辑
```typescript
// 一级分类点击
const handleCategoryClick = (categoryId: string) => {
  setSelectedCategory(categoryId);
};

// 二级分类点击
const handleSubCategoryClick = (subCategoryId: string) => {
  setSelectedSubCategory(subCategoryId);
};
```

#### 智能分类同步
```typescript
useEffect(() => {
  if (currentCategory && currentCategory.subCategories.length > 0) {
    // 检查当前二级分类是否有效
    const isValidSubCategory = currentCategory.subCategories.some(
      sub => sub.id === selectedSubCategory
    );
    // 只在无效时才重置
    if (!isValidSubCategory) {
      setSelectedSubCategory(currentCategory.subCategories[0].id);
    }
  }
}, [selectedCategory, currentCategory, selectedSubCategory]);
```

### 3. 条件渲染

#### 动态样式
```typescript
className={`px-4 py-2 rounded-full text-sm font-medium transition-colors ${
  selectedSubCategory === subCategory.id
    ? 'bg-blue-500 text-white'           // 选中状态
    : 'bg-gray-200 text-gray-700 hover:bg-gray-300'  // 默认状态
}`}
```

#### 空状态处理
```typescript
{currentTools.length === 0 && (
  <div className="text-center py-12">
    <p className="text-gray-500">该分类下暂无工具</p>
```
